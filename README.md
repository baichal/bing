# 微软Bing 必应积分自动脚本 (积分检测版) 使用说明

## 📖 脚本简介
这是一个专为 Microsoft Rewards（必应积分）设计的自动化辅助脚本。它会在必应搜索页面生成一个悬浮窗，通过搜索全网“今日热榜”的真实关键词来模拟人类搜索行为。

**核心特色**：脚本内置了**智能积分检测机制**。只有当您的必应积分**实际增加**时，脚本才会将此次搜索计入“已完成次数”。这能有效防止因网络延迟、未登录或达到每日上限导致的“空刷”和计数虚高问题。

---

## 🌟 主要功能

1.  **💰 智能计分**
    *   **实时检测**：每次搜索后自动比对积分变化。
    *   **有效计数**：只有积分涨了，进度条才会+1。
    *   **自动跳过**：如果积分未变（例如今日分数已拿满），脚本不会增加计数，而是直接搜索下一个词，直到达成您设定的有效次数目标。

2.  **🔥 真实热词搜索**
    *   接入“今日热榜”数据（微博、知乎、百度等），使用真实的实时热点关键词。
    *   **自动换榜**：当前榜单关键词搜完后，脚本会自动切换到下一个平台的榜单继续执行，无需人工干预。

3.  **🖥️ 现代化悬浮窗**
    *   **随意拖拽**：按住标题栏即可将窗口拖动到屏幕任意位置，刷新不重置。
    *   **最小化模式**：点击右上角 `−` 可收起窗口，仅保留小标题栏，不遮挡网页内容。
    *   **深色模式适配**：跟随系统或浏览器主题自动切换颜色。

4.  **🛡️ 安全防封**
    *   **随机延迟**：每次搜索间隔随机设定为 **8~14 秒**，模拟真实用户浏览节奏。

5.  **⏯️ 断点续传**
    *   停止脚本或关闭网页后，脚本会记住当前的搜索进度。下次打开时，会从上一次停止的关键词接着搜，不会从头重复。

---

## 🛠️ 使用步骤

1.  **准备工作**：安装 Tampermonkey (油猴) 扩展，并安装本脚本。
2.  **进入页面**：打开 [Bing.com](https://www.bing.com) 并进行一次任意搜索。
3.  **检查登录**：确保右上角已登录 Microsoft 账号，且能看到积分数字（如 1234）。
4.  **启动脚本**：
    *   页面右侧会出现悬浮窗。
    *   输入您想完成的**有效搜索次数**（默认 50 次）。
    *   点击 **“开始”** 按钮。
5.  **挂机等待**：
    *   脚本显示“自动搜索中... 等待 X 秒”即为正常运行。
    *   观察红色计数器，积分涨一次，数字加 1。

## ❓ 常见问题 (Q&A)

**Q: 为什么脚本在自动搜索，但“有效搜”的次数（红色数字）没有增加？**
**A:** 这是本脚本的智能保护机制。
*   脚本检测到您的积分**没有增加**。
*   原因可能是：网络延迟、未登录、或者**您今天的搜索积分上限已经拿满了**。
*   脚本会继续搜下一个词，直到检测到积分为止。如果长时间不涨，请检查是否已达今日上限。

**Q: 点击“开始”后一直显示等待？**
**A:** 为了模拟真人，脚本设有 8-14 秒的随机冷却时间。请耐心等待倒计时结束，它会自动跳转。

**Q: 搜索完一个榜单后脚本停了吗？**
**A:** 没有。如果当前榜单（如微博）的词搜完了，但还没达到您设定的次数，脚本会自动切换到下一个榜单（如知乎）继续搜索。
